## Setup

- **Terminal:** Ghostty + tmux
- **Shell:** fish + starship
- **Editor:** nvim
- **Source control:** git + gh + tig + lazygit
- **Theme:** tokyonight

## Neovim

- **Plugin manager:** lazy
- **Statue line:** lualine
- **Completions:** nvim-cmp
- **LSP:** mason, none-ls
- **Syntax highlighting:** treesitter
- **Fuzzy finder:** telescope
- **File tree:** nvim-tree
- **Snippets:** LuaSnip
- Other goodies...

## Setup

1. Run `./setup`
2. Login to 1Password and enable the SSH Agent under "Developer"
3. Start Raycast and import settings under `dotfiles/raycast/backups` (the password is in 1Password under www.raycast.com -> Backup Password)
